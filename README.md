Pharmacy Inventory Automation Pipeline
Overview
A custom automation workflow designed to streamline inventory data synchronization between Google Sheets and hospital survey systems. This project minimizes manual data entry errors and optimizes hospital workflow efficiency.

Key Features
Seamless Integration: Direct sync from Google Sheets to the survey interface.

Dynamic Data Processing: Handles diverse pharmaceutical datasets using XPath selection.

Error Handling: Built-in verification logic to prevent data mismatch during the survey entry process.

Technologies
Tool: java script

Data Source: Google Sheets API

Automation Logic: XPath/CSS Selectors for DOM interaction

How to use
Install Automa extension.

Import the workflow.json file.

Update the Spreadsheet ID and credentials in the "Get Range" block.
